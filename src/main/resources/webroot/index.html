青鲨超级浏览器

一款能够支持网页浏览，控件可视化，视频展示，在线网游等不同功能的综合客户端软件

Github

  • 支持多种格式定义页面元素：JSON，XML 等

    <xmlv title="样例展示">
    <json>
    [{"type":"button","id":"button001","x":10,"text":"click me"}]
    </json>
</xmlv>

  • 支持多种语言：Javascript，Java，Ruby，Groovy，Latte等

    <xmlv title="样例展示">
    <json>
    [{"type":"button","id":"button001","x":10,"text":"click me"}]
    </json>
    <script type=”javascript”>
        button001.action = function(event){
        	button001.text = “我被点啦”
        }
    </script>
</xmlv>

  • 支持传统的 HTTP服务器：您只需要将HTML格式文件改成XML(V)格式

    <xmlv title="样例展示">
    <json>…</json>
    <script type="javascript">…</script>
</xmlv>

  • 以及更多有趣的功能：例如动画以及游戏制作：

    var gameloop = app.timer
var moving_button = false
gameloop.action = function(event){
	if(moving_button)
		button001.x += 1
}
button001.action = function(event){
	moving_button = !moving_button
}
gameloop.start()
